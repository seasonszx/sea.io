• 数据库采用
    MS SQL SERVER 2005

    对于开发人员来说，最常见的就是操作数据表，那么你首先必须创建好数据表
    同时注意几点建表规则

    在线演示
    相关数据库操作文档下载
    1. 数据表名大写，必须有中文说明
    2. 字段名大写必，须有中文说明
    3. 字段ID不能修改，必须为主键且自增，步长为1
    4. 数据类型支持int,bigint,smallint,decimal,
      varchar(N)此外其他类型视为varchar类型
    5. 非空属性会影响添加，修改页面的必填项
    6. 长度属性会影响输入框的输入长度
    支持← →方向键

  • 启动X-code代码生成器
    目前只支持MS SQL SERVER数据库
    其他数据库支持后续将会依次和大家见面
    敬请大家关注

    1. 必须在项目lib文件夹下有你选择的数据库的jar包
    2. 数据库驱动字符串会和数据库自动匹配
    3. 链接账号和密码输入完成后建议先测试是否链接成功

  • 选择库和表
    选择你项目对于的数据库
    选择你要生成实体类对于的数据表
    无脑点击下一步

    1. 结构简单
    2. 操作快捷
    3. 一目了然

  • 配置参数
    配置生成文件的部分属性
    目前版本已支持统计图，自定义流程表单

    相关配置详细说明文档下载
    1. 列表页面的搜索项,排序及生成路径等参数
    2. 配置是否需要生成统计和自定义流程

  • 生成文件
    生成文件包含两类：.java文件和.jsp文件
    java文件：bean(实体类),dao(接口),daoimpl(接口实现类),
    sql(数据库操作语句),action(控制跳转)

    1. 基础jsp：getList,goAdd,goUpd,goView
    2. 统计jsp：goStatics
    3. 流程jsp：processList,goStartTask,
      goDoingTask,goViewTask,goViewForm
